### 1. What is Linux?
Linux is a free and open source operating system that anyone can use, study, modify and distribute. It was originally created by Linus Torvalds in 1991 as a hobby project and has since grown into one of the most widely used operating systems in the world. Today Linux powers web servers, cloud infrastructure, smartphones, supercomputers and cybersecurity tools.

### 2. Why is Linux important in Cyber Security?
Linux is the most important operating system in cybersecurity because nearly all security tools like Nmap, Metasploit, Wireshark, Burp Suite and John the Ripper are built to run on Linux. Most servers and cloud systems run Linux too, so security professionals need to understand it to know how to protect or attack these systems. Distributions like Kali Linux are specifically designed for penetration testing and ethical hacking with hundreds of tools pre-installed.

### 3. Difference between Linux and Windows

| Feature | Linux | Windows |
|---------|-------|---------|
| Cost | Completely free and open source | Requires a paid license |
| Security | Highly secure, very few viruses | Frequently targeted by malware |
| Customisation | Fully customisable at every level | Limited customisation options |
| Primary use | Servers, cybersecurity, development | Home and office desktop use |
| Terminal | Extremely powerful command line | Basic command prompt |
| Privacy | Does not collect user data | Collects telemetry and user data |

### 4. What is a Linux Distribution?
A Linux distribution (distro) is a complete operating system built on top of the Linux kernel but packaged with different software, tools, desktop environments and default settings depending on its purpose. The core kernel is the same but the experience varies widely. For example Kali Linux is built for cybersecurity professionals, Ubuntu is designed for everyday desktop use, and CentOS is designed for enterprise servers.

### 5. Why do Ethical Hackers prefer Linux?
Ethical hackers prefer Linux for several reasons. It gives complete control over the system with no restrictions. It has a powerful terminal where complex tasks can be automated using scripts. Kali Linux comes with over 600 pre-installed security and hacking tools. Linux is also lightweight, stable and rarely crashes. Since most target systems like web servers and IoT devices run Linux, working in the same environment makes testing much more effective and realistic.
